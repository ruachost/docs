## Dashboard Walkthrough

This guide mirrors the platform workflow from login to final send, including contact management, sender ID usage, scheduling, and reports.

### 1) Open SMS Module and Compose Screen

Start on the dashboard, open the `SMS MT` module, and click `Compose SMS`.

![Dashboard overview](/images/dashboard-overview.png)
![Dashboard overview (variant)](/images/dashboard-overview001.png)
![Compose SMS screen](/images/send-sms.png)

### 2) Enter Campaign Details

Provide a campaign name and sender ID before adding recipients.

![Campaign name field](/images/Click-Campaign-name-to-name-the-campaign.png)
![Sender ID input](/images/Input-your-SenderID.png)
![Manage Sender IDs menu](/images/manage-senderIDs.png)

### 3) Prepare Contacts

You can type numbers directly, import from file, or select from existing groups.

![Contacts menu](/images/contacts.png)
![Manage groups entry](/images/contacts-02.png)
![Manage groups table](/images/contacts-03.png)
![Add group modal](/images/contacts-04.png)
![Bulk import action from group](/images/contacts-05.png)
![Import contacts step 1](/images/contacts-06.png)
![Import contacts step 2](/images/contacts-07.png)
![Contacts input area](/images/Input-your-contacts.png)

### 4) Configure Message Settings

Choose encoding, compose your message, and optionally use Flash SMS.

![Data coding selector](/images/data-coding.png)
![Message content input](/images/message-content.png)

### 5) Optional: Schedule Message

If you do not want immediate delivery, enable scheduling before sending.

![Schedule checkbox](/images/schedule-message.png)

### 6) Send Message

Click `Send`, review the message preview and cost breakdown, then confirm.

![Compose screen with send button](/images/send-sms-001.png)
![Message preview confirmation](/images/send-sms-002.png)

### 7) Track Delivery and Download Reports

Use `Reports` to monitor status, filter by date range, and export results.

![Campaign report menu](/images/report.png)
![Campaign report list view](/images/report02.png)

### 8) Monitor Wallet / Credit

Check wallet/credit balance to ensure sufficient funds before large campaigns.

![Wallet screen](/images/Wallet.png)
