## Get Campaigns

Retrieve a paginated list of SMS campaigns with optional date filtering.

<CodeGroup>

```bash cURL
curl -X GET "https://accounts.netvox.ng/api/v2/Campaign?ApiKey=YOUR_API_KEY&ClientId=YOUR_CLIENT_ID&start=0&length=10&fromdate=2024-01-01&enddate=2024-12-31" \
  -H "Content-Type: application/json"
```

```javascript JavaScript
const params = new URLSearchParams({
  ApiKey: 'YOUR_API_KEY',
  ClientId: 'YOUR_CLIENT_ID',
  start: '0',
  length: '10',
  fromdate: '2024-01-01',
  enddate: '2024-12-31'
});

const response = await fetch(`https://accounts.netvox.ng/api/v2/Campaign?${params}`, {
  method: 'GET',
  headers: {
    'Content-Type': 'application/json'
  }
});

const data = await response.json();
console.log(data);
```

```python Python
import requests
from datetime import datetime

url = "https://accounts.netvox.ng/api/v2/Campaign"
params = {
    "ApiKey": "YOUR_API_KEY",
    "ClientId": "YOUR_CLIENT_ID",
    "start": 0,
    "length": 10,
    "fromdate": "2024-01-01",
    "enddate": "2024-12-31"
}

response = requests.get(url, params=params)
data = response.json()
print(data)
```

</CodeGroup>

## Parameters

<ParamField query="ApiKey" type="string" required>
  Your API key for authentication
</ParamField>

<ParamField query="ClientId" type="string" required>
  Your client identifier for authentication
</ParamField>

<ParamField query="start" type="number" required>
  Starting index for pagination (0-based)
</ParamField>

<ParamField query="length" type="number" required>
  Number of campaigns to retrieve per page
</ParamField>

<ParamField query="fromdate" type="string">
  Start date filter in `yyyy-mm-dd` format
</ParamField>

<ParamField query="enddate" type="string">
  End date filter in `yyyy-mm-dd` format
</ParamField>

## Response

<ResponseField name="ErrorCode" type="number">
  Error code (0 for success)
</ResponseField>

<ResponseField name="ErrorDescription" type="string">
  Description of the result
</ResponseField>

<ResponseField name="Data" type="array">
  Array of campaign objects
</ResponseField>

<ResponseField name="Data[].CampaignId" type="number">
  Unique identifier for the campaign
</ResponseField>

<ResponseField name="Data[].CampaignName" type="string">
  Name of the campaign
</ResponseField>

<ResponseField name="Data[].Message" type="string">
  Campaign message content
</ResponseField>

<ResponseField name="Data[].CreatedDate" type="string">
  Campaign creation date
</ResponseField>

<ResponseField name="Data[].SenderId" type="string">
  Sender ID used for the campaign
</ResponseField>

<ResponseField name="Data[].MessageCount" type="number">
  Number of messages in the campaign
</ResponseField>
