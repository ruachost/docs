## Get Campaign Statistics

Retrieve detailed delivery statistics and performance metrics for a specific campaign.

<CodeGroup>

```bash cURL
curl -X GET "https://accounts.netvox.ng/api/v2/Campaign/Statistics?ApiKey=YOUR_API_KEY&ClientId=YOUR_CLIENT_ID&campaignId=2272" \
  -H "Content-Type: application/json"
```

```javascript JavaScript
const params = new URLSearchParams({
  ApiKey: 'YOUR_API_KEY',
  ClientId: 'YOUR_CLIENT_ID',
  campaignId: '2272'
});

const response = await fetch(`https://accounts.netvox.ng/api/v2/Campaign/Statistics?${params}`, {
  method: 'GET',
  headers: {
    'Content-Type': 'application/json'
  }
});

const data = await response.json();
console.log(data);
```

```python Python
import requests

url = "https://accounts.netvox.ng/api/v2/Campaign/Statistics"
params = {
    "ApiKey": "YOUR_API_KEY",
    "ClientId": "YOUR_CLIENT_ID",
    "campaignId": 2272
}

response = requests.get(url, params=params)
data = response.json()
print(data)
```

</CodeGroup>

## Parameters

<ParamField query="ApiKey" type="string" required>
  Your API key for authentication
</ParamField>

<ParamField query="ClientId" type="string" required>
  Your client identifier for authentication
</ParamField>

<ParamField query="campaignId" type="number" required>
  Campaign ID (obtained from Get Campaigns API)
</ParamField>

## Response

<ResponseField name="ErrorCode" type="number">
  Error code (0 for success)
</ResponseField>

<ResponseField name="ErrorDescription" type="string">
  Description of the result
</ResponseField>

<ResponseField name="Data" type="object">
  Campaign statistics object
</ResponseField>

<ResponseField name="Data.DeliveredCount" type="number">
  Number of successfully delivered messages
</ResponseField>

<ResponseField name="Data.DNDCount" type="number">
  Number of messages blocked due to Do Not Disturb
</ResponseField>

<ResponseField name="Data.SuccessCount" type="number">
  Number of successful submissions
</ResponseField>

<ResponseField name="Data.SubmittedCount" type="number">
  Total number of submitted messages
</ResponseField>

<ResponseField name="Data.UnDeliveredCount" type="number">
  Number of undelivered messages
</ResponseField>

<ResponseField name="Data.RejectedCount" type="number">
  Number of rejected messages
</ResponseField>

<ResponseField name="Data.TempSubmittedCount" type="number">
  Number of temporarily submitted messages
</ResponseField>

<ResponseField name="Data.OthersCount" type="number">
  Number of messages with other statuses
</ResponseField>

<ResponseField name="Data.DroppedCount" type="number">
  Number of dropped messages
</ResponseField>

<ResponseField name="Data.TotalCount" type="number">
  Total number of messages in the campaign
</ResponseField>

<ResponseField name="Data.NotProcessedCount" type="number">
  Number of unprocessed messages
</ResponseField>

## Message Status Definitions

| Status | Description |
|--------|-------------|
| **Delivered** | Message successfully delivered to recipient |
| **DND** | Message blocked due to Do Not Disturb list |
| **UnDelivered** | Message failed to deliver |
| **Rejected** | Message rejected by carrier or system |
| **Dropped** | Message dropped due to various reasons |
| **Not Processed** | Message not yet processed |

## Use Cases

- **Campaign analytics**: Track campaign performance and delivery rates
- **ROI analysis**: Calculate success rates and optimize future campaigns
- **Compliance monitoring**: Monitor DND and rejection rates
- **Reporting**: Generate detailed campaign reports for stakeholders
