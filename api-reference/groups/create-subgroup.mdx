## Create Sub-Group

Create a sub-group under an existing parent group for hierarchical organization.

<CodeGroup>

```bash cURL
curl -X POST "https://accounts.netvox.ng/api/v2/Group/33/SubGroup" \
  -H "Content-Type: application/json" \
  -d '{
    "GroupName": "Sub Group Name",
    "ApiKey": "YOUR_API_KEY",
    "ClientId": "YOUR_CLIENT_ID"
  }'
```

```javascript JavaScript
const response = await fetch('https://accounts.netvox.ng/api/v2/Group/33/SubGroup', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    GroupName: 'Sub Group Name',
    ApiKey: 'YOUR_API_KEY',
    ClientId: 'YOUR_CLIENT_ID'
  })
});

const data = await response.json();
console.log(data);
```

```python Python
import requests

url = "https://accounts.netvox.ng/api/v2/Group/33/SubGroup"
data = {
    "GroupName": "Sub Group Name",
    "ApiKey": "YOUR_API_KEY",
    "ClientId": "YOUR_CLIENT_ID"
}

response = requests.post(url, json=data)
result = response.json()
print(result)
```

</CodeGroup>
